    SDK预览

    欢迎来到 Android Things SDK 预览！Android Things开发和传统的Android移动开发非常相似。它也通过使用Android框架和工具进行app的编写。 你所需准备的仅仅是一台刷写了Android Things OS的开发板，以及其他开发必须使用的外设。

    这篇指南将帮助你快速上手开发所支持的开发板，同时也将指导你如何设置初始开发环境。

    初识Android开发

    如果你从未开发过Android应用程序，那么咱们先从构建你的第一个Android移动应用开始说起。因为这其中包含了开发Android Things所涉及的 一些基础概念和核心开发流程。如果你还没有一台移动设备，那么我们可以先从软件模拟器开始尝试Android SDK。

    查看 Android OS文档中的 构建你的第一个应用程序 来帮助你开始Android Thingks开发。准备好了吗？Let's Go！

    理解Android Things平台

    因为Android Things 和Android OS核心还是有些许关键点的不同，请参阅 概览 来理解这些你必须知晓的核心概念。

    获取硬件

    在你开始之前，你需要一台支持Android Things的开发板。你可以在 开发套件页面 查看这些可用的开发板。

    烧录Android Things

    一旦你选择了一块开发板，请参考硬件入门指南中的说明为你的的开发板进行烧录和设置。

    • Intel Edison
    • Intel Joule
    • NXP Pico
    • Raspberry Pi 3

    设置你的开发环境

    1. 下载 或者将你的Android Studio升级至最新版本。
    2. 导入 项目模板 例程到你的Android Studio。

    3. 连接你的开发板并通过adb验证你的设备: 

      $ adb devices
List of devices attached
4560736843791520041    device

    4. 部署 示例程序到你的开发板并使用logcat验证你所看到的活动信息。

    构建你的第一台设备

    现在你的环境已经设置好了，请参考 构建你的第一台设备 来训练数据，并开始Android Things开发。

    更多示例

    为了帮助构建更为复杂的Android Things应用，请在 示例 部分参阅其他样例
